# 🎰 ZooloCASINO - Resultados Oficiales

Sistema de resultados de lotería de animales con soporte para Venezuela y Perú.

## ✨ Características

- 📅 Resultados por fecha
- 🇻🇪 ZooloCASINO Venezuela (12 sorteos)
- 🇵🇪 ZooloCASINO Perú (11 sorteos)
- 🎲 Triple ZooloCASINO
- 🔢 Terminal ZooloCASINO
- ⭐ Más 1 (sorteo especial viernes)
- 📱 Panel de administración con calendario
- 🖼️ 82 imágenes de animales caricaturescos

## 🛠️ Tecnologías

- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui
